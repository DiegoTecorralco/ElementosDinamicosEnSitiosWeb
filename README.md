# Elementos Dinamicos en Sitios Web
Actividad de aprendizaje práctica para desarrollar un sitio web interactivo con animaciones, transiciones, eventos JavaScript y carrusel de imágenes, mejorando la experiencia de usuario.

# Objetivo de la Actividad
El estudiante desarrollará un sitio web interactivo completo, implementando los elementos dinámicos más utilizados en el desarrollo web moderno. Al finalizar, contarás con un proyecto real en tu portafolio.

- **Eventos del Mouse:** Efectos hover dinámicos que responden al cursor del usuario.
-  **Eventos de Scroll:** Mostrar y ocultar elementos al desplazarse por la página.
-  **Manipulación del DOM:** Controlar secciones y menús con botones interactivos.
- **Carrusel de Imágenes:** Galería automática o manual con mínimo 3 imágenes.
